# Программа для переиспользования видео-контента с помощью LLM

## Поднятие сервера LM Studio

Для работы программы необходима программа LM Studio.

В LM Studio необходимо установить модель в соответствии с характеристиками своего ПК

После установки модели необходимо поднять сервер LM Studio с помощью команды:

```bash
lms server start
```

## Запуск программы

Для установки всех зависимостей необходимо выполнить команду:

```bash
uv sync
```

Для запуска программы необходимо указать название модели, ранее установленной в LM Studio:

```bash
uv run -m llm_content_recylcer <название_модели>
```

Например:

```bash
uv run -m llm_content_recylcer qwen3-4b-thinking-2507
```

После запуска команды по адресу http://127.0.0.1:7861 поднимется веб-интерфейс.

Библиотека gradio помимо локальной ссылки создают публичную, однако при тестировании программы
публичная ссылка была не рабочая.
